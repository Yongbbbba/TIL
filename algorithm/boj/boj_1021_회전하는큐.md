# boj 1021 회전하는 큐

- 스스로 힘으로 풀지 못함. 처음에는 백트래킹으로 풀어야 하는게 아닌가 싶었는데 애초에 불가능한건지 구현을 할 줄 모르는건지 실패함
- 풀이 방법은 find로 현재 입력 원소의 위치를 파악한 후에 push_back이 유리한지 push_front가 유리한지 판단하여 풀이 

## 코드

```c++
#include <bits/stdc++.h>

using namespace std;

#define endl '\n'
#define ll long long

int main()
{
	ios::sync_with_stdio(0);
	cin.tie(0);
	int n, m;
	int result = 0;
	cin >> n >> m; 

	deque<int> dq; 
	for (int i = 1; i <= n; i++)
	{
		dq.push_back(i);
	}

	while (m--)
	{
		int num; 
		cin >> num; 
		
		// num의 위치
		int left = find(dq.begin(), dq.end(), num) - dq.begin();

		while (dq.front() != num)
		{
			int right = dq.size() - left;
			if (left < right) // pop_front가 유리
			{
				dq.push_back(dq.front());
				dq.pop_front();
			}
			else
			{
				dq.push_front(dq.back());
				dq.pop_back();
			}
			result++;
		}
		dq.pop_front();
	}
	cout << result;

		return 0;
}
```
